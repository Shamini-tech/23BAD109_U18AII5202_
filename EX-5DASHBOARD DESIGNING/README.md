## Objective
To design an interactive analytical dashboard for monitoring key academic and operational metrics in real time.
## Scenario
An AI-based Smart Campus System requires a centralized dashboard to track and analyze student attendance, academic performance, and resource utilization effectively.
## Tools Used
Microsoft Power BI Desktop, campus dataset, and visualization components.
## In-Lab Tasks Performed
### 1. KPI Card Design
Designed KPI cards to display overall attendance percentage and academic performance metrics for quick monitoring.
### 2. Filtering and Slicing
Implemented interactive filters based on department and semester to allow customized analysis of data.
### 3. Linked Views and Drill-Down Analysis
Connected multiple visuals and enabled drill-down features to analyze data from summary level to detailed level.
## Features of the Dashboard
Interactive interface, real-time analysis, cross-filtering, department-wise insights, and easy navigation.
## Outcome
The dashboard helps administrators monitor key metrics, analyze trends, and make informed decisions using visual analytics.
